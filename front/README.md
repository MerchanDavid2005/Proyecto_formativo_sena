# Bienvenido a mi proyecto formativo #

*Bienvenido a mi proyecto formativo, es una pagina web de compras en linea de componentes de todo tipo de vehiculo, en las que podras realizar tu carrito de compras y hacer pedidos*

## ¿Pero que puedo hacer exactamente? ##

* Ver los diferentes productos que hay, tambien podras buscarlos y filtrar por sus categorias

* Podras seleccionar varios productos y añadirlos a tu carrito de compras 

* Tambien podras ver los diferentes servicios que ofrece la empresa
